# Avzaman-Voron-Switchwire (Big Pink)
Voron Switchwire build for @avzaman

## Hardware
Printer: Voron Switchwire [(LDO Rev.A)](https://docs.ldomotors.com/en/voron/voronsw/printed_part_guide_rev_a)\
Toolhead: Voron Stealthburner\
Extruder: Voron Clockwork2\
Hotend: Phaetus Dragonfly BMO - 0.4mm Brass Nozzle

Bed Probe: [Klicky Probe](https://github.com/jlas1/Klicky-Probe/tree/main/Printers/Voron/Switchwire)

Motors XYZ: LDO-42STH40-1684AC - 1.8° Nema 17 Stepper\
Motor E: Moons CSE14HRA1L410A-01 - 1.8° Nema 14 Stepper

## Electronics
MCU: Orange Pi 4 LTS V1.2\
Mainboard: BTT SKR Mini E3 V2.0\
PSU: Meanwell LRS-350-24V\
PSU: Meanwell RS-25-5V\
Toolhead Board:[ BTT EBB SB2209 CAN V1.0](https://global.bttwiki.com/EBB%202240%202209%20CAN.html)\
CANBus Board: BTT U2C V2.1\
LCD: mini12864

## Filament
Polymaker ASA - Black\
Polymaker ASA - Pop Pink
